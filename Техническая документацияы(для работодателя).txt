Техническое описание проекта "PC Shot Gun Honor"

Обзор проекта:
"PC Shot Gun Honor" — многопользовательская игра, созданная в Unity, вдохновлённая механикой захвата точек из "For Honor", но с использованием винтовки из "Hunt: Showdown". В игре участвуют две команды (красная: 1 игрок + 2 бота, белая: 3 бота), которые борются за контроль над двумя точками (A и B) и уничтожение противников. Проект фокусируется на сложном ИИ ботов и динамичной игровой механике.

Техническая реализация:
- Версия Unity: 22.3.37f1
- Язык программирования: C#
- Игровая механика:
  - Захват точек: Реализована система триггеров (Collider в Unity) для определения присутствия игрока/бота на точке. Логика захвата включает таймер, который прерывается при появлении врага.
  - Бонусная система: Счётчик очков для каждой команды, увеличивающийся при удержании точек. При достижении 150 очков противники теряют возрождения (реализовано через ограничение спавна).
  - Оружие: Винтовка из "Hunt: Showdown" смоделирована с использованием анимаций и физики Unity (Raycast для стрельбы, Rigidbody для попаданий).
  - Спавн и возрождение: Система спавна с заданными точками появления, отключаемая для проигравшей команды при достижении порога бонусов.
- ИИ ботов:
  - Реализован с использованием конечных автоматов для управления поведением.
  - Боты отслеживают состояние точек (захвачены/захватываются), выбирают приоритетные цели (захват, атака, патрулирование).
  - Логика отступления: Боты оценивают уровень здоровья и отступают к безопасной зоне или запрашивают помощь.
  - Группировка и помощь: Реализована система сообщений между ботами для координации (например, запрос помощи при низком здоровье).
  - Патрулирование: Боты перемещаются по маршруту, которые сами себе строют, если нет активных задач.
- Интерфейс: Минималистичный UI с отображением очков, статуса точек и здоровья, построенный на Unity UI.
- Сетевые аспекты: Игра рассчитана на локальный режим (без сетевого мультиплеера), но ИИ имитирует поведение реальных игроков.

Проблемы, с которыми столкнулся:
- Разработка ИИ: Создание сложного ИИ ботов было наиболее трудоёмкой задачей. Требовалось сбалансировать их поведение, чтобы они были достаточно умными, но не подавляли игрока. Решено путём настройки весов приоритетов задач.
- Балансировка: Настройка скорости захвата точек и порога бонусных очков (150) потребовала нескольких итераций тестирования для достижения сбалансированного геймплея.
- Производительность: Оптимизация ИИ (особенно группировки и патрулирования) для минимизации нагрузки на процессор, так как 5 ботов одновременно выполняют сложные вычисления.
- Анимации оружия: Адаптация винтовки из "Hunt: Showdown" требовала точной настройки анимаций и таймингов стрельбы для реалистичного ощущения.

Уроки, вынесенные из проекта:
- Сложный ИИ требует тщательного тестирования и отладки, чтобы избежать непредсказуемого поведения.
- Балансировка динамики захвата точек критически важна для создания увлекательного геймплея.
- Использование готовых ассетов (анимации, модели) ускоряет разработку, но требует доработки для соответствия стилю игры.

Возможные улучшения:
- Добавить поддержку сетевого мультиплеера для игры с реальными игроками.
- Расширить арсенал оружия, добавив другие виды из "Hunt: Showdown".
- Улучшить визуальные эффекты для захвата точек и стрельбы.
- Добавить больше точек захвата или динамические события на карте.
